# 🍔 Food Truck Menu – React Components


---

## 📌 Live Link
https://food-truck-menu-nine.vercel.app/

---

## 📖 Project Overview
A React-based food truck menu system designed to replace manual paper/Excel-based operations with a fast, reliable, and structured UI.

---

## 🎯 Features
- Fast and responsive UI
- Clean component-based architecture
- Real-time user interaction handling
- Structured menu rendering.

---

## ⚠️ Edge Case Handling
- Empty state → "No data found"
- Loading state → Loader for slow network
- Invalid input → Form validation with error highlight.

---

## 🧠 Technical Stack
- React (Vite)
- useState, useEffect
- Prop drilling only
- Monochrome UI design.

---

## ♿ Accessibility & Security
- Keyboard navigable UI
- ARIA labels included
- Input sanitization (XSS protection)

---

## ✅ Done Criteria
- No errors in build
- No ESLint warnings
- All edge cases handled
- Fully accessible UI
